¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado principal para crear páginas web. Define la estructura del contenido usando elementos representados por etiquetas.

<!DOCTYPE html>
<html>
  <head>...</head>
  <body>...</body>
</html>

Elementos HTML (tipos y ejemplos)

Un elemento HTML es la unidad básica de la página: puede ser una etiqueta de apertura y cierre, o una etiqueta auto-cerrada. Los elementos pueden clasificarse por su comportamiento y propósito:

  • Elementos de bloque: ocupan todo el ancho disponible y acostumbran a formar la estructura (por ejemplo, <div>, <p>, <h1>).
  • Elementos inline: ocupan solo el espacio necesario y suelen estar dentro de otros elementos (por ejemplo, <span>, <a>, <img>).
  • Elementos semánticos: describen el propósito del contenido y mejoran la accesibilidad y SEO (por ejemplo, <header>, <nav>, <section>, <article>, <footer>).

Ejemplos rápidos:

<!-- Elemento de bloque (estructura) -->
<div>
  <h2>Sección ejemplo</h2>
  <p>Párrafo dentro de un <div>.</p>
</div>

<!-- Elementos inline dentro de texto -->
<p>Texto con <strong>énfasis</strong> y un <a href="https://example.com">enlace</a>.</p>

<!-- Elemento semántico -->
<article>
  <h3>Artículo interesante</h3>
  <p>Contenido del artículo...</p>
</article>

Consejo: usar elementos semánticos mejora la lectura por parte de lectores de pantalla y motores de búsqueda. Si no existe un elemento semántico adecuado, <div> y <span> siguen siendo útiles como contenedores neutros.

Elementos y etiquetas

Un elemento HTML normalmente tiene una etiqueta de apertura y una de cierre. Ejemplo:

<p>Este es un párrafo.</p>

Algunas etiquetas son auto-cerradas, por ejemplo la etiqueta de imagen:

<img src="ruta.jpg" alt="Descripción" />

Atributos

Los atributos añaden información a las etiquetas. Aparecen dentro de la etiqueta de apertura.

<a href="https://ejemplo.com" target="_blank">Visitar sitio</a>

Títulos y párrafos

Los títulos van de <h1> (más importante) a <h6> (menos importante). Los párrafos usan <p>.

<h1>Título principal</h1>
<p>Un párrafo explicativo.</p>

Enlaces e imágenes

Enlaces con <a> e imágenes con <img>:

<a href="https://www.example.com">Ir a example.com</a>
<img src="https://via.placeholder.com/150" alt="Ejemplo" />

Listas

Listas ordenadas (<ol>) y no ordenadas (<ul>):

<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>

Formularios

Formularios recogen datos del usuario. Ejemplo simple:

<form action="/enviar" method="post">
  <label>Nombre: <input name="nombre" /></label>
  <button type="submit">Enviar</button>
</form>
