Aprendiendo HTML Básico

Una guía práctica con ejemplos interactivos

1. Estructura Básica HTML

Todo documento HTML debe comenzar con la declaración DOCTYPE y tener elementos html, head y body:

<!DOCTYPE html> <html> <head> <title>Título de la página</title> </head> <body> El contenido va aquí </body> </html>
Nota: La declaración DOCTYPE le dice al navegador que esto es un documento HTML5.

2. Encabezados HTML

HTML tiene seis niveles de encabezados, desde h1 (el más importante) hasta h6:

<h1>Encabezado 1</h1> <h2>Encabezado 2</h2> <h3>Encabezado 3</h3> <h4>Encabezado 4</h4> <h5>Encabezado 5</h5> <h6>Encabezado 6</h6>

3. Párrafos y Formato de Texto

Los párrafos se definen con la etiqueta p. El formato de texto incluye b, strong, i, em, mark, small, sub, sup:

<p>Este es un párrafo normal.</p> <p> <b>Texto en negrita</b> y <strong>texto importante</strong> <br> <i>Texto en cursiva</i> y <em>texto enfatizado</em> <br> <mark>Texto resaltado</mark> y <small>texto pequeño</small> </p>

4. Enlaces e Imágenes

Los enlaces se crean con la etiqueta a y las imágenes con img:

<a href="https://www.ejemplo.com">Visitar ejemplo.com</a> <br> <img src="https://via.placeholder.com/150" alt="Imagen de ejemplo" width="150" height="150">
Visitar ejemplo.com
Importante: Siempre incluye el atributo alt en las imágenes para mejorar la accesibilidad.

5. Listas

HTML soporta listas ordenadas (ol) y no ordenadas (ul):

<h3>Lista no ordenada:</h3> <ul> <li>Café</li> <li>Té</li> <li>Leche</li> </ul> <h3>Lista ordenada:</h3> <ol> <li>Primer paso</li> <li>Segundo paso</li> <li>Tercer paso</li> </ol>

Lista no ordenada:

  • Café
  • Té
  • Leche

Lista ordenada:

  1. Primer paso
  2. Segundo paso
  3. Tercer paso

6. Tablas HTML

Las tablas se crean usando las etiquetas table, tr, th y td:

<table border="1"> <tr> <th>Encabezado 1</th> <th>Encabezado 2</th> </tr> <tr> <td>Fila 1, Celda 1</td> <td>Fila 1, Celda 2</td> </tr> <tr> <td>Fila 2, Celda 1</td> <td>Fila 2, Celda 2</td> </tr> </table>

7. Formularios HTML

Los formularios se crean con la etiqueta form y pueden contener diferentes tipos de inputs:

<form action="/enviar" method="post"> <label for="nombre">Nombre:</label> <input type="text" id="nombre" name="nombre"> <br><br> <label for="email">Email:</label> <input type="email" id="email" name="email"> <br><br> <input type="submit" value="Enviar"> </form>




Nota: Este formulario es solo demostrativo y no enviará datos a ningún servidor.
